# Functions-Beta

#### 介绍
Minecraft Plugins

#### 特点
    
    登陆功能√
    获取玩家地址√
    用户组√
    服务器√
    - Tab(列表栏)√
    - ScoreBoard(计分板)√
    - ActionBar(物品栏上方显示)√
    - JsonText(原版tellraw显示)√

#### 安装教程

    下载最新版本√
    放入 你的服务器目录/plugins √
    (如果服务器在运行重启)开启服务器√
    生成配置文件√
    手动修改配置信息√
    如果你的ip查询没有更新请到网络上寻找更新√
